Requirements
------------

SqueezePHPAPI requires PHP version 5, which was released in July 2004. 
If your provider still uses PHP 4, then ask them politely to upgrade.


Installation
------------

SqueezePHPAPI can be simply downloaded at http://org.0r9.org


Upgrading
---------

Probably just replace the whole library.


Using SqueezePHPAPI
-------------------

The SqueezePHPAPI is a set of developer classes. Just check the documentation
for usage.



Changing SqueezePHPAPI
----------------------

If you find a bug in SqueezePHPAPI then please send a report back so that it can
be fixed in the next version.  You can submit your bugs and other
requests here:

http://org.0r9.org
